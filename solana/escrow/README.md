# wager-escrow (Anchor)

Deterministic program id:
- `solana/escrow/scripts/PROGRAM_ID.txt`
- keypair: `solana/escrow/target/deploy/wager_escrow-keypair.json`

## Toolchain
- Solana CLI installed (`solana --version`)
- Anchor installed (`anchor --version`)

## Run localnet tests

```bash
cd solana/escrow
yarn install
anchor test
```
